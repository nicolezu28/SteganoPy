# SteganoPy 🔐

Aplicație de steganografie care îți permite să ascunzi mesaje secrete și fișiere în imagini.

## Instalare

```bash
pip install streamlit pillow numpy cryptography
```

## Rulare

```bash
streamlit run app.py
```

Aplicația se va deschide automat în browser la `http://localhost:8501`

## Funcționalități

- 📝 Encodare/decodare mesaje text în imagini
- 📁 Ascundere/extragere fișiere binare în imagini  
- 🔐 Criptare opțională AES-256
- 📦 Procesare în lot (batch) pentru multiple imagini
